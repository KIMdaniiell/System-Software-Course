# 2.8 Жадные квантификаторы

**Квантификатор** - конструкция, которая позволяет указывать количество повторений
+ **{n}** - *n* раз
+ **{m,n}** - от *m* до *n*
+ **{m,}**
+ **{,n}**
+ **?** - ноль или одно повторение (есть или нет) = **{0,1}**
+ **\*** - сколько угодно повторений = **{0,}**
+ **+** - как минимум 1 повторение = **{1,}**
